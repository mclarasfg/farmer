Conectar Google Calendar
Use o Google Apps Script — gratuito, sem Cloud Console, sem cartão de crédito. Você já tem a URL? Cole direto abaixo.
Cole sua URL do Apps Script aqui
Se você já criou o script e tem a URL (começa com https://script.google.com/macros/s/), cole abaixo e clique em Conectar.
Ainda não tem a URL? Siga os 4 passos abaixo

Abrir o Apps Script

Código para colar no Apps Script
function doGet() {
  var cal = CalendarApp.getDefaultCalendar();
  var passado = new Date();
  passado.setMonth(passado.getMonth() - 12);
  var futuro = new Date();
  futuro.setMonth(futuro.getMonth() + 3);

  var eventos = cal.getEvents(passado, futuro);
  var resultado = eventos.map(function(e) {
    return {
      titulo: e.getTitle(),
      data: Utilities.formatDate(
        e.getStartTime(),
        Session.getScriptTimeZone(),
        "yyyy-MM-dd"
      ),
      descricao: e.getDescription() || ""
    };
  });

  return ContentService
    .createTextOutput(JSON.stringify(resultado))
    .setMimeType(ContentService.MimeType.JSON);
}
Como usar o Farmer Dashboard
Guia completo para gestão da sua carteira de clientes — do cadastro ao acompanhamento.
1 Primeiros passos
Configure seu perfil
Clique no seu avatar no canto inferior esquerdo da sidebar. Insira seu nome e função. Seu nome vai aparecer no cabeçalho e nos relatórios exportados.
Importe seus clientes existentes
Se você já tem uma planilha de clientes, clique em Importar planilha no topo da tela. A planilha precisa seguir o modelo padrão (disponível para download com quem te enviou este arquivo).
A importação não duplica clientes com o mesmo nome. Você pode importar várias vezes sem problema.
Cadastre um novo cliente
Clique em + Novo cliente no canto superior direito. Preencha o nome, data de início da consultoria, frequência de reuniões após a C4, e as datas das consultorias obrigatórias que já aconteceram.
2 Entendendo o card do cliente
Aba Plano
Contém tudo sobre o acompanhamento do cliente:
  • Consultorias C1–C4 — obrigatórias. Clique direto na data para editar. Verde = realizada, amarelo = agendada (futura), vermelho = não realizada.
  • Acompanhamentos pós-C4 — reuniões de manutenção após o ciclo inicial. Adicione quantas quiser clicando em "+ Adicionar". Clique na data para editar ou na lixeira para remover.
  • Aportes mensais — valor que o cliente transfere mensalmente para a conta investimento. Registre mês a mês para acompanhar a evolução. Use a lixeira para corrigir um valor errado.
  • Produtos — clique em qualquer produto para alternar entre Ativo, Potencial e —.
Aba Tarefas
Anote tudo que precisa fazer em relação ao cliente — cotar seguro, cobrar S1NC, agendar revisão.
  • Digite a tarefa, escolha a categoria e defina um prazo (opcional).
  • Pressione Enter ou clique no + para adicionar.
  • Clique no checkbox para marcar como concluída. Ela vai para "Concluídas" sem desaparecer.
  • Tarefas atrasadas ficam em vermelho, as que vencem hoje em amarelo.
O badge roxo no card fechado mostra quantas tarefas estão pendentes. Clique nele para abrir direto na aba de tarefas.
Aba Notas
Campo livre para anotações sobre o cliente — contexto de vida, observações de reunião, informações relevantes. Salva automaticamente quando você clica fora do campo.
3 Alertas e indicadores de risco
O sistema calcula automaticamente o risco de cada cliente com base em quatro critérios:
  • C1, C2, C3 ou C4 não realizadas — consultoria obrigatória sem data registrada.
  • C4 agendada mas ainda não ocorreu — data futura, ciclo ainda aberto.
  • Tempo sem reunião — dias desde a última reunião superou a frequência definida no cadastro (ex: cliente mensal sem reunião há mais de 30 dias).
  • Sem aporte no mês atual — nenhum aporte registrado no mês corrente.
Saudável — sem alertas Atenção — 1+ alerta leve Risco — C obrigatória ausente
4 Visões globais
Todas as tarefas
Clique em Todas as tarefas na sidebar (ou no card de métricas) para ver todas as tarefas pendentes da carteira em uma tela só — sem precisar abrir cliente por cliente. Filtre por categoria ou ordene por prazo, cliente ou tipo.
Potenciais de produto
Clique no card Potenciais de produto nos métricas do topo. Abre um painel mostrando, para cada produto, quais clientes foram marcados como "Potencial" — útil para planejar abordagens comerciais em lote.
5 Google Calendar
Conecte sua agenda Google para preencher datas automaticamente. Clique em Google Calendar na sidebar e siga os 4 passos para criar o script.
Importante: para o sistema encontrar o cliente certo, o título do evento na sua agenda precisa conter o nome completo do cliente. Exemplo: "C2 - João Silva" ou "Reunião Ana Marques".
Após conectar, clique em Sincronizar agora sempre que quiser atualizar. O sistema preenche as datas de C1–C4 ainda vazias e adiciona reuniões de acompanhamento automaticamente.
6 Exportar e gerenciar
Exportar relatório
Clique em Exportar no topo. Abre um arquivo HTML com o resumo visual da sua carteira — status, datas e tarefas de todos os clientes. Tem botão "Imprimir / Salvar PDF" direto na página.
Desativar ou excluir cliente
Clique nos três pontos (⋮) no canto direito do card do cliente.
  • Desativar — o cliente fica acinzentado e marcado como inativo, mas os dados são mantidos. Útil para clientes em pausa. Pode reativar a qualquer momento.
  • Excluir — remove permanentemente todos os dados (tarefas, notas, histórico). Pede confirmação antes.
7 Dúvidas frequentes
Os dados ficam salvos onde?
No navegador do seu computador (localStorage). Isso significa que os dados ficam na máquina onde você abriu o arquivo — não vão para nenhum servidor, não precisam de internet. Se abrir em outro computador ou navegador, a carteira estará vazia.
Posso perder meus dados?
Sim, se limpar os dados do navegador (histórico, cookies, cache). Por isso, exporte o relatório periodicamente como backup, e use a planilha de importação para guardar uma cópia dos seus clientes fora do navegador.
Dois consultores podem compartilhar a mesma carteira?
Não nesta versão — cada arquivo é individual. Cada consultor tem o seu arquivo com sua própria carteira no seu próprio navegador. Para compartilhar dados entre pessoas, use a exportação + importação de planilha.
Como corrigir uma data errada?
Na aba Plano do cliente, clique direto na data — ela vira um campo editável. Mude e clique fora para salvar. Para aportes, use a lixeira ao lado do mês incorreto e registre novamente com o valor certo.
